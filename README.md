# Todolist-client

Client react for web, with [client-core](https://gitlab.com/todolist-micro-services/todolist-client-core.git)

## Run

```bash
yarn && yarn dev
```

### Docker
```bash
make # -> Unix or MacOS system
### or
docker-compose -f ./config/docker/docker-compose.yml up --build -d
```
